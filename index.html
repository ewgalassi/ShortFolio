Counter Stock Game

Counter Stock is a game that allows a user to create characters with a stock portfolio. The character initially starts with a value of $1000, but the value changes depending upon the percent change of the stock in the current market day. This character can then battle other characters belonging to different users. Victory results in increases in the portfolio value, while losses will decrease its value. This game uses API calls, HTML, CSS, JavaScript, localStorage, JQuery, Bootstrap, Handlebars, Express, and Node.
